自定义 CSS

自定义 CSS 叠加在当前主题之上，通过 <style> 注入到 Bilibili 页面。 它与主题解耦：切换或删除主题都不会改动它。

启用

自定义 CSS 说明

时钟节点带 bpx-player-top-clock 类，子元素按布局形态命名为 bpx-player-clock-*。以下选择器覆盖所有内置布局形态。

必须用 !important。 内置样式是 inline style（style="…"），优先级高于普通 CSS 规则。 要覆盖字号、颜色、透明度、动画等 inline 属性，必须加 !important，否则不会生效。

可用选择器

.bpx-player-top-clock时钟容器（所有布局都有）
.bpx-player-clock-single-time基础数字 / 胶囊布局的数字
.bpx-player-clock-segment分舱数码的每一段数字舱
.bpx-player-clock-rec-dot录像时间码的红点
.bpx-player-clock-rec-label录像时间码的 "REC" 字样
.bpx-player-clock-rec-time录像时间码的时间数字
.bpx-player-clock-corner-left边角框的左侧框
.bpx-player-clock-corner-right边角框的右侧框
.bpx-player-clock-corner-time边角框中间的时间数字
.bpx-player-clock-analog-dial指针表盘的圆盘
.bpx-player-clock-analog-tick指针表盘的刻度
.bpx-player-clock-analog-marker指针表盘的 12/3/6/9 数字
.bpx-player-clock-analog-hand指针表盘的指针（共 3 根）
.bpx-player-clock-analog-pin指针表盘的中心轴
.bpx-player-clock-flip-card翻页时钟的每一格翻牌
.bpx-player-clock-hud-tag科幻 HUD 的 "TIME" 标签
.bpx-player-clock-hud-time科幻 HUD 的时间数字
.bpx-player-clock-calendar-header日历桌牌的日期条
.bpx-player-clock-calendar-time日历桌牌的时间数字

示例

点击「复制」可直接粘贴到上面的输入框试用。

脉冲呼吸动画 
@keyframes biclock-pulse {
  0%, 100% { opacity: 1; }
  50%      { opacity: 0.55; }
}

.bpx-player-top-clock {
  animation: biclock-pulse 2s ease-in-out infinite !important;
}
霓虹光晕（青色） 
.bpx-player-top-clock {
  text-shadow:
    0 0 0.18em rgba(94, 234, 212, 0.9),
    0 0 0.42em rgba(94, 234, 212, 0.6) !important;
}
隐藏录像时间码的红点 
.bpx-player-clock-rec-dot {
  display: none !important;
}
更换数字字体 
.bpx-player-top-clock {
  font-family: "DSEG7 Classic", ui-monospace, monospace !important;
}

关于

BiClock 是一个轻量的 Manifest V3 浏览器扩展，在 Bilibili 视频播放器顶部叠加显示当前时间。 纯原生 JS，零依赖、零构建，同一套代码同时支持 Chrome / Edge / Firefox。
